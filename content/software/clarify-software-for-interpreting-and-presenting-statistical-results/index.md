summary: |-
  This is a set of easy-to-use tools that implement the techniques described in Gary King, Michael Tomz, and Jason Wittenberg's "Making the Most of Statistical Analyses: Improving Interpretation and Presentation." Winner of the Okidata Best Research Software Award from the American Political Science Association. These tools use Monte Carlo simulations to compute interpretable quantities from regression models and perform inference on them. For Stata, see the Journal of Statistical Software article (doi:10.18637/jss.v008.i01); for current R implementations, see https://iqss.github.io/clarify

<div class="hwp-text-block field field--name-field-hwp-body field--type-text-long field--label-hidden"><p>This is a set of easy-to-use tools that implement the techniques described in Gary King, Michael Tomz, and Jason Wittenberg's "<a href="/gking-site/files/abs/making-abs.shtml">Making the Most of Statistical Analyses: Improving Interpretation and Presentation</a>". Winner of the <em>Okidata Best Research Software Award</em> from the American Political Science Association. These tools use Monte Carlo simulations to compute interpretable quantities from regression models and perform inference on them.</p><h3>{clarify} for R</h3><ul><li>Implements predictions at representative values, average marginal effects, and any user-specified quantities of interest in a simulation framework, as well as visualization methods. {clarify} for R represents an evolution of the {<a href="https://zeligproject.org/">Zelig</a>} R package by restoring and adding to simulation-based functionality for translating hard-to-interpret coefficients into meaningful quantities of interest. </li><li>Authors: Noah Greifer, Steven Worthington, Stefano Iacus, and Gary King.</li><li>Website: <a href="https://iqss.github.io/clarify">https://iqss.github.io/clarify</a></li><li>GitHub: <a href="https://github.com/iqss/clarify">https://github.com/iqss/clarify</a></li><li>CRAN page: <a href="https://cran.r-project.org/package=clarify">https://cran.r-project.org/package=clarify</a></li><li>See website for installation instructions, documentation, and examples.</li><li>Provides functionality previously provided by {Zelig}; see instructions on website for converting a {Zelig}-based workflow to one that uses {clarify} instead.</li></ul><h3>clarify for Stata</h3><ul><li>Implements predictions at representative values and visualization methods in a simulation framework.</li><li>Authors: Michael Tomz, Jason Wittenberg, and Gary King.</li><li>Github: <a href="https://github.com/iqss-research/clarify">https://github.com/iqss-research/clarify</a> </li><li>Installation instructions and documentation are provided in a JSS Paper: <ul><li>Tomz, Michael, Jason Wittenberg, and Gary King. 2003. “Clarify: Software for Interpreting and Presenting Statistical Results.” Journal of Statistical Software 8: 1–30. <a href="https://doi.org/10.18637/jss.v008.i01">https://doi.org/10.18637/jss.v008.i01</a></li></ul></li><li>A user donated wrapper from Fred Wolfe is available to automate clarify's simulation of dummy variables and can be installed with: ssc install qsim</li></ul></div>
