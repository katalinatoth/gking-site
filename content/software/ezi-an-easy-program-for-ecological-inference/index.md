Previous versions and information about the software are archived here.</em></p><p>This is a stand-alone, menu-oriented version of EI that runs under Windows. It does not require Gauss or any other software to run.</p><p>EzI does everything EI does and with fewer startup costs but, due to the lack of the Gauss command line, is somewhat less flexible.
