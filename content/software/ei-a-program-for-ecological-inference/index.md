<div class="hwp-text-block field field--name-field-hwp-body field--type-text-long field--label-hidden"><p>This program provides easy-to-use methods of running all the statistical procedures, diagnostics, and graphics developed in <a href="/gking-site/publication/a-solution-to-the-ecological-inference-problem-reconstructing-individual-behavior-from-aggregate-data/">A Solution to the Ecological Inference Problem: Reconstructing Individual Behavior from Aggregate Data</a> (Princeton University Press, 1997). The program has been rewritten from scratch in R: <a href="https://github.com/iqss-research/ei">eiR on GitHub</a>. The older Gauss version is still available here.</p><ul><li><strong>Reporting Bugs and Issues: </strong>Please use our Github Issue <a href="https://github.com/iqss-research/ei/issues/new">form.</a><br/> </li><li><strong>Questions and feature requests:</strong> Discuss the software on our Discussions <a href="https://github.com/iqss-research/ei/discussions">page</a>.<br/> </li><li><strong>EI for Gauss:</strong><ul><li>To install, see the <a href="https://github.com/iqss-research/ei">eiR repository on GitHub</a></li><li><span>Documentation:</span><a data-entity-substitution="canonical" data-entity-type="node" data-entity-uuid="0a44c39d-52a4-4c1d-9c8c-63be57c6ce08" href="#" title="ei.pdf">PDF</a></li><li>EzI version (standalone executable): <a href="/gking-site/software/ezi-an-easy-program-for-ecological-inference/">Website</a>.<br/> </li></ul></li><li><strong>EI for R:</strong> See <a href="https://github.com/iqss-research/ei">the eiR repository on GitHub</a>.</li></ul></div>
