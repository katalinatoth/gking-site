<h2 style="font-size:1.15rem;font-weight:700;margin:1.25rem 0 0.5rem;color:#111;">Abstract</h2>
<div style="padding:14px 16px;background:#ebf2f8;border-radius:6px;border:1px solid #d0dce8;">
<p style="margin:0;">In this paper, we present an overview of the Virtual Data Center (VDC) software, an open-source digital library system for the management and dissemination of distributed collections of quantitative data. (See <a href="https://dataverse.org/" style="color:#337ab7;">Dataverse</a>.) The VDC functionality provides everything necessary to maintain and disseminate an individual collection of research studies, including facilities for the storage, archiving, cataloging, translation, and on-line analysis of a particular collection. Moreover, the system provides extensive support for distributed and federated collections including: location-independent naming of objects, distributed authentication and access control, federated metadata harvesting, remote repository caching, and distributed “virtual” collections of remote objects.</p>
