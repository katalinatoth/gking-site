<div class="hwp-text-block field field--name-field-hwp-body field--type-text-long field--label-hidden"><p>Authors: <a href="http://www.stanford.edu/%7Etomz/">Michael Tomz</a>, <a href="/gking-site/">Gary King</a>, <a href="mailto:langche@everest.fas.harvard.edu">Langche Zeng</a></p><p>Both versions implement the suggestions described in Gary King and Langche Zeng's "<a href="/gking-site/files/abs/0s-abs.shtml">Logistic Regression for Rare Events Data</a>", "<a href="https://Logistic Regression in Rare Events Data">Explaining Rare Events in International Relations</a>" and "<a href="g/files/abs/1s-abs.shtml">Estimating Risk and Rate Levels, Ratios, and Differences in Case-Control Studies </a>". Options for density case-control sampling designs are, at present, only available in the Gauss version.</p><p>Stata code is available at <a href="https://github.com/iqss-research/relogit">https://github.com/iqss-research/relogit</a>.</p><ul><li><strong>Updated versions of ReLogit for R</strong> are available as part of the comprehensive statistical package <a href="#">Zelig: Everyone's Statistical Software</a>.  Zelig runs within <a href="http://www.r-project.org/">R</a> on all commonly used hardware and operating systems.</li><li><strong>Reporting Bugs and Issues: </strong>Please use our Github Issue <a href="https://github.com/iqss-research/relogit/issues/new">form.</a><br/> </li><li><strong>Questions and feature requests:</strong> Discuss the software on our Discussions <a href="https://github.com/iqss-research/relogit/discussions">page</a>.</li><li><strong>Installing Relogit for Stata:</strong><ul><li>Install from SSC by typing <span>ssc install relogit</span>.</li><li>To install manually, download the package from the link below, and then put the files in the 'plus' directory, under 'r'. To find that directory, type <span>sysdir.</span></li><li>For documentation, type <span>help relogit</span></li><li>A helpful package to graph predictive probabilities and confidence intervals from <span>relogitq</span> is available <a href="https://github.com/aliatia-1/relogitplot">here</a>.</li></ul></li></ul></div>
