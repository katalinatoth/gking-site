# EditMe/Misc/

Standalone pages that have a URL on the live site but don't fit any of
the menu sections.

Examples:

- `accessibility/` — accessibility statement.
- `advice/` — Gary's advice for grad students.
- `aiwrite/` — AI writing tools.
- `apply/` — application info.
- `commencement-speech/` — commencement speech text.
- `electronic-collection/` — research data collection.
- `miscellanea/` — miscellaneous notes.
- `papers/` — older paper landing page.
- `pnas-edit/` — PNAS editorial.
- `recs/` — recommendation-letter info.
- `ssa/` — social-security-administration page.

These pages are reachable via direct URL (e.g.,
<https://gking.harvard.edu/accessibility/>) but aren't linked from the
top menu.
