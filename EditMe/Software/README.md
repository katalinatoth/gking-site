# EditMe/Software/

Software packages Gary has authored or contributed to. One folder per
package.

```
Software/
├── _index.md                           the /software/ list page
├── <package-slug>/index.md             one folder per software package
│                                       (Amelia, EI, Clarify, Zelig, …)
└── Data/                               data files for the software list
    └── software_*.yaml
```

Each `<package-slug>/` folder contains the package's `index.md`
(description, links, citation), an icon or screenshot, and any
supplementary files.
