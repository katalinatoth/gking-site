```
docs/

├── CLAUDE_PROMPTS.md        copy-paste AI prompts for common tasks
├── REPO_LAYOUT.md           plain-language tour of every top-level folder

├── WEBSITE_PRINCIPLES.md    architecture playbook for building a similar site

└── audits/                  individual audit reports (CLASSIFY_REVIEW.md, etc.)
```

The top-level [`README.md`](../README.md) at the repo root is the entry
point and links into this folder.
