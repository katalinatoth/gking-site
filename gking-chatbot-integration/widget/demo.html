This page includes the Gary King chat widget as a single <script> tag. The floating launcher should appear in the bottom-right corner.

Important — do NOT open this file via file://

The widget makes cross-origin requests to AWS API Gateway, which strips CORS response headers when the request Origin is null (browsers send null for file:// URLs). Serve this directory over HTTP instead — for example: 

cd $(dirname this-file)
python3 -m http.server 8000
# then open http://localhost:8000/

On Gary's actual HTTPS site this is a non-issue — the origin will be a real https://... URL and CORS will work transparently.
